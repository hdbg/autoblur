## Setup
- Set your own infura url in `fakemask.js`
- Create `config.toml` with following fields:
    - `pk` your private key; String
    - `min_top_bids` minimum bids at top to track; Number
    - `slugs` list of tracked/bidding collections; Array

- `cargo run`
